# AUTO3016 - 自动控制实践 A 实验

![成绩构成](https://img.shields.io/badge/成绩构成-gold) ![实验报告90%](https://img.shields.io/badge/实验报告-90%25-wheat) ![答辩10%](https://img.shields.io/badge/答辩-10%25-wheat)

课程的相关任务见 `materials` 文件夹下的文件 `2024_自动控制实践A综合实验.pdf`。如要参考实验的详细思路，请见 `labs/2023/WJD` 文件夹下的答辩 ppt。
这门综合实验要求**两人一组**，制作一块有刷直流电机驱动板，包含 PWM 发生模块（TL494）、功放模块（L298）、相对旋转编码器（测电机转速）、电流采集芯片（ACS712）。
注：课程的实际情况和 ppt 上写的有些不同的地方，需要提前拷问老师从而尽早做安排（如编码器的参数表）。
自控实践 A 期末考试实际上并不包含综合实验的内容；综合实验的报告并不需要当堂交，什么时候交都可以（除了给分上可能略有差异）。

  - 暂无详细评价。

这门课是本科期间与硬件最为相关的一门课程了，但是做出来的成品含金量只有大二甚至大一学生的水准。
对于从来没有接触过硬件设计的同学，这门课程姑且相当于**入门**，但除了一个任务说明 ppt，没有给学生**相应的任何指导**，学生只得自力更生。
2023 及以前叫课程设计，2024 开始叫实验。从名字的更改也能看出，其实并没有什么设计的感觉，因为选型都是老师决定好的，所有的报告和答辩，其实都只是在帮老师阐释这个选型的合理性。
用的接线方式是在下发的 PCB 板上飞杜邦线。说是为了锻炼学生设计能力，实则一是完全不符合工程做法，二是完全没有设计感。

> 文 / [Oliver Wu](https://www.github.com/OliverWu515), 2025-01-01

不要因为这门课电路设计简单，而对从事硬件工程师工作产生不切实际的幻想。实际硬件设计需考虑的东西要复杂得多。
> 文 / [Oliver Wu](https://www.github.com/OliverWu515), 2025-01-01

虽然选型没有要求，但对于电路结构、参数的设计还是有一些要求。
2023 年有许多同学在设计 TL494 电路时，完全不会阅读手册，甚至很清晰的典型应用也没读明白，最后靠抄别人电路通过；不关注模块前后关系，输出的电平等级直接烧毁后级的 L298。
建议大家，**如果要参考学长学姐的设计，要确保先阅读相关芯片的 datasheet，对实验原理有基本的理解，方便后续排查**。
测量电路好坏时，要有顺序和步骤，一般先查接线（断电用通断档），再上电查电源，再查一些确定性的信号。

> 文 / [Oliver Wu](https://www.github.com/OliverWu515), 2025-01-01

一般会分成若干教室上课，每个教室都会有老师和助教。遇到问题建议优先寻求网络、然后再找助教。推荐上半岛小芯查 datasheet。
注意测量电机机械特性时转矩的数量级。

> 文 / [Oliver Wu](https://www.github.com/OliverWu515), 2025-01-01
