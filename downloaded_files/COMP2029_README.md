# COMP2029 - 面向领域的计算机系统设计与开发实践

![成绩构成](https://img.shields.io/badge/成绩构成-gold) ![过程检查30%](https://img.shields.io/badge/过程检查-30%25-wheat) ![结题验收70%](https://img.shields.io/badge/结题验收-70%25-wheat)

2018 级大三下、2019 级大四上开设 COMP2024B 软件设计与开发实践B Design and Practice of software (B)。
2020 级起改为大四上开设 COMP2029 面向领域的计算机系统设计与开发实践 Domain Oriented for System Design and Practice。


课程以导师课题分组带领学生的形式授课，需要在选课环节先选导师，进行学生导师互选。
课程可以做毕设相关内容，毕设导师不在授课教师名单内也可以联系老师进行答辩验收结课。
