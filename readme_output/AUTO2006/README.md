# 控制理论中的代数基础

**课程代码:** AUTO2006

## 考试

**2025 春考试情况：**
*   **考试难度**：中。
*   **题型**：15 个判断题 30 分，九道大题 70 分。题目包括但不限于：课本课后习题原题、作业题魔改、书本有关定理证明。
*   **内容**：大多为基础题、计算题，拔高的构造式证明题仅占 3 分。
*   **总结**：由于是考查课，同学态度总体不算认真，复习不充分导致成绩不理想。希望同学们以思维锻炼的眼光看待这门课。

> 文 / [Oliver Wu]([https://github.com/OliverWu515](https://github.com/OliverWu515)), 2025.7

## 建议

1.  **预习建议**：建议在课程开始前（如寒假）阅读《线性代数及其应用》(Lay D.C.) 的前四章。这本书比较浅显易懂，适合建立直观认知。国内教材通常计算较多，不注重线性空间，因此推荐以此书作为真正的第一次学习。
2.  **上课建议**：前半部分相对较新，需注意理解作业题思路。如果预习好了，后半部分学习会比较轻松。

> 文 / [AutoFriedRich]([https://github.com/AutoFriedRich](https://github.com/AutoFriedRich)), 2024.5

本课侧重于**线性空间**的知识，与大一的代数与几何（重矩阵运算）不同，强调“弱计算，重思想”。
课程使用**英文 PPT**，中文讲授；作业和考试要求用英语完成，对英语水平有一定要求（掌握常用词汇术语即可）。
课程架构与 *Linear Algebra Done Right* 不完全一致，有的重点内容书上没有，建议提前下载老师讲义预习。

> 文 / [Maxwell Jay & longlin li]([https://github.com/MaxwellJay256](https://github.com/MaxwellJay256)), 2023.6
