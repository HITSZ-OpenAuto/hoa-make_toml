# 自动控制理论B

**课程代码:** AUTO3001B

## 考试

*   **难度**：中等。
*   **特点**：
    *   20/21级情况：考试时间较紧，题型传统，不考深难题。
    *   2025年情况：考得很简单，无花样，高分段密集。
*   **重要提示**：老师上课强调「不会考试」的点真的不考。

> 文 / Oliver Wu & Maxwell Jay, 2025.8

## 建议

自动化专业看家本领，内容稍显繁琐但不复杂。
想读研的同学，在学状态空间时应打好线性代数基础，涉猎《线性系统理论》。
频域法部分对考研同学较为有用。

> 文 / [Oliver Wu]([https://www.github.com/OliverWu515](https://www.github.com/OliverWu515)), 2024.11

**自控 B 的特点**：
1. 结果不唯一
2. 存在近似处理
3. 使用经验公式
4. 设计过程具有主观性

**重点掌握**：
频域校正、离散校正、能控能观分解、状态空间综合、非线性系统描述函数法。这些都有标准公式和流程，多做作业题即可掌握。

> 文 / [Maxwell Jay]([https://www.github.com/MaxwellJay256](https://www.github.com/MaxwellJay256)), 2025.8
