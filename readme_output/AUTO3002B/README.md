**Costannt 总结的重点（仅供参考）：**
1.  **元件部分**：直流/交流电机公式及调速；测量元件原理；各类元件优缺点。
2.  **设计部分**：
    *   背诵控制系统设计完整流程及各步骤作用。
    *   输入信号分析结果的应用（DFT/FFT计算非重点）。
    *   **误差系数**：静态/动态误差系数推导，两种系数关系，跟踪误差计算优缺点。
    *   **噪声**：谱密度、相关函数关系，均方误差，等效噪声带宽。
    *   **抑制方法**：全部误差抑制方法，特别是扰动观测器中低通滤波器的设计。
    *   **系统特性**：灵敏度函数，Bode积分定理，对象不确定性，鲁棒稳定性条件，机械谐振（自由转子模型）。
    *   **抗饱和**：Windup现象及Anti-windup设计。
    *   **系统类型**：伺服系统（校正方法）、调节系统（设计原则）、过程控制系统（整定方法）。
    *   **多回路**：设计矛盾、优缺点、串级调节、复合控制。
3.  **智能控制**：
    *   专家控制（23级未涉及重点）。
    *   模糊控制：优缺点、流程、关键点。

## 实验

**本部实验安排（23级）：**

**实物实验（共16学时）：**
1.  **电机位置伺服系统模型辨识**：测试频率特性，辨识传递函数模型。
2.  **电机位置伺服系统频域控制器实验验证**：频域校正实现，闭环性能测试。
3.  **PID+干扰观测器控制器设计与仿真验证**：测试扰动响应，设计干扰观测器。
4.  **多回路控制器实验验证**：位置、速度、电流多回路控制，调试准则与性能测试。

**仿真实验（共16学时）：**
5.  **倒立摆的模糊自适应PID控制器设计**：建模仿真，不确定性分析，模糊自适应PID实现。
6.  **伺服系统模糊控制设计与分析**：建模仿真，构建规则，设计模糊控制器及参数优化。
