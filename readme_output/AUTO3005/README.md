## 实验

**实验安排：**

| 序号 | 教学内容 | 教学要求 |
| :--- | :--- | :--- |
| 1 | 工业机器人的示教编程与焊接 | 认知商用系统，示教编程完成基本运动及焊接实验。 |
| 2 | SCARA/并联/六轴机器人运动学 | 求解并编程完成上述机器人的前向运动学控制。 |
| 3 | 正逆运动学实验 | 利用 MATLAB 求解六轴机器人逆运动学并仿真，可选实机验证。 |
| 4 | 机器人运动规划与控制 | 完成基本运动规划仿真，并进行实机实验分析结果。 |
