## 建议

虽然选型没有要求，但对于电路结构、参数的设计还是有一些要求。
2023 年有许多同学在设计 TL494 电路时，完全不会阅读手册，甚至很清晰的典型应用也没读明白，最后靠抄别人电路通过；不关注模块前后关系，输出的电平等级直接烧毁后级的 L298。
建议大家，**如果要参考学长学姐的设计，要确保先阅读相关芯片的 datasheet，对实验原理有基本的理解，方便后续排查**。
测量电路好坏时，要有顺序和步骤，一般先查接线（断电用通断档），再上电查电源，再查一些确定性的信号。

> 文 / [Oliver Wu]([https://www.github.com/OliverWu515](https://www.github.com/OliverWu515)), 2025.1

一般会分成若干教室上课，每个教室都会有老师和助教。遇到问题建议优先寻求网络、然后再找助教。推荐上半岛小芯查 datasheet。
注意测量电机机械特性时转矩的数量级。
