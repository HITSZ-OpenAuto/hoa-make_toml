# 凸优化与最优控制

**课程代码:** AUTO5023

## 考试

允许携带一张A4纸。可以带计算器（但基本用不上）。

这门考试的考点其实是很明确的，因为所讲的内容确实不多。凸优化中的复杂证明在课上就不会讲到，而是被当作“对我们没有什么用处”的“纯粹的数学”一笔带过（笑）。而最优控制问题中的复杂计算又难以考查。考试中最难的也就是一些稍微需要技巧的证明（主要是优化问题之间的转化），如果之前没有了解具体方法未必能想出证法，不过好在这些在课上都会讲到。
