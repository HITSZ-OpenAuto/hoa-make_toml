## 考试

（注：以下为原来考试课时代的情况）
期末考试难度整体适中，打分较友好，最高分 93 左右，平均分 80 左右。挂科率大约在百分之一到百分之二。
考试主要考查各种知识的概念，主要是简答题，要记背的东西非常多！！
考试复习时，建议以背诵 PPT 为主，能背多少背多少，同时也可以将老师给的主观题答案背下来。

（注：以下为原来考试课的情况）
