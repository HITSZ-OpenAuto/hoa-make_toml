# 计算机组成原理

**课程代码:** COMP2008

## 考试

**2020年及以前**：
类似于考研课，范围明确，死板，搞懂课堂练习、教材例题和作业即可。

**2021年**：
题量很大，考查很多概念。新增 RISC-V 部分很灵活，题目难度较高。

**2022年**：
题量大，RISC-V 是重点。理解 PPT 内容后不会有太意外的题目。
大致题型回忆：
- 10 个选择，15 个填空。
- 简答题：浮点数二进制表示、八大思想（选三个描述）、总线传输周期、汉明码计算、RAM 和 ROM 的扩展连接。
- 补码计算（2小题）。
- Cache 全相联、组相联、直接映射和命中率计算。
- RISC-V 小填空（指令含义、表示、汇编结果）。
- 数据冒险分析和代码重排序。
- 处理器设计分析（类似作业大题）。

> 文 / 往届学生, 2023.6

## 实验

实验难度较大，需要使用 Verilog HDL。
建议：
1. 认真温习相关知识，理解变量、模块意义、时钟沿等。
2. Debug 时耐心看波形图。
3. 抱紧有经验的助教或大腿同学。
4. 实验内容每年会有变化，平均耗时 1 晚至 2 天。
心态上不要抵触 HDL，否则会很痛苦。
