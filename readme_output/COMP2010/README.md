## 课程评价

课程是很难的课程，涉及的概念都挺抽象的，但是过还是很容易的。

## 考试

18、19 级考得很简单，作业能搞懂或者背下来就没问题了。
选择填空基本上是概念题，计算题也是很基础的，证明题在课上有暗示过可能会考，因此可以提前复习。
