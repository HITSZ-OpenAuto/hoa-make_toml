## 考试

想不挂科是不难的，不用担心挂科。考试范围和上课的范围相比要小很多（老师会给出考试范围），不要求记忆公式和算法流程（但还是考了贝叶斯公式，不过也只考了这一个），无计算，有一些**送分题**。
但是，**有的选项PPT中根本没有**。复习时建议参考“模式识别-2022级期末复习指南”，里面按知识点对2023秋的期末试题进行了整理。
