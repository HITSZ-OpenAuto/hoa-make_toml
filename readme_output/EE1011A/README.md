## 其他

### 学习顺序建议

我个人会将第二、三章的学习顺序调整如下：
2.3 支路电流法；2.4 回路电流法；2.5 节点电压法；2.6-2.7 含运放电路的分析；2.1 电阻网络等效；3.1/2.2/3.3 含源支路和含源一端口等效；3.2 齐性定理、叠加定理；3.4-3.6 特勒根定理、互易定理、对偶原理。

此外，预习时可将第六章（非正弦周期电流电路）提到第五章（三相电路）之前。
