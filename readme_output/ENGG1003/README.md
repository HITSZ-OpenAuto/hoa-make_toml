## 课程评价

**23级（2024夏）学时安排和上课内容详情：**

1. **绪论（6学时）**
   - 内容：三次讲座，介绍重要性、现代工业发展及项目管理。
   - *评价*：前两次与实验关系不大；第三次涉及图表，与最后项目制作报告有关。

2. **元器件识别（4学时）**
   - 内容：识别常见元件，使用万用表、LCR、晶体管图示仪。
   - *评价*：简单，照着指导书做即可。需绘制特性曲线。

3. **电子封装（4学时）**
   - 内容：了解扩晶、固晶、键合等流程，熟悉机器使用。
   - *评价*：纯体验，设备昂贵由老师指导，无实验报告。

4. **电子可靠性（4学时）**
   - 内容：静电打击芯片并观察损坏。
   - *评价*：静电高压注意安全。实验蛮好玩的，可以体验击穿芯片的快感。

5. **硬件设计（8学时）**
   - 内容：设计温度报警器，使用 OrCAD Capture 仿真及嘉立创 EDA 绘图。
   - *评价*：仿真实验建议无脑按照指导书设计即可。

6. **PCB设计（8学时）**
   - 内容：利用嘉立创 EDA 绘制电路 PCB 图。
   - *评价*：没有基础会有一点痛苦，建议找人带。本步骤设计的东西不会被实际加工，后续组装使用老师设计好的板子。

7. **焊接调试（8学时）**
   - 内容：焊接组装小怪兽温度报警器并测试。
   - *评价*：最痛苦的实验。焊贴片和芯片很难，且实验室电烙铁保养极差，经常只有一点能加热。

8. **软件设计（20学时）**
   - 内容：利用 MSP430F5529 及扩展板实现 GPIO、PWM、ADC、SPI 等功能。
   - *评价*：学过嵌入式的直接上手，没学过的改示例代码。代码记得保存，后续项目制作会大量复用。

9. **项目制作（14学时）**
   - 内容：实现温度报警、音乐报警、呼吸灯、风扇调速、墨水屏/OLED显示等功能。
   - *评价*：PWM 和屏幕显示可能有冲突。板子性能问题可能产生各种 bug，切记不要红温。建议找大佬带以速通。

10. **项目发布（4学时）**
    - 内容：PPT 成果汇报及实物展示。
    - *评价*：PPT 需包含任务分配、甘特图等，注意美观。

## 建议

1. 虽然项目很简单，但对于无嵌入式和硬件基础的人来说独立完成不容易。
2. 建议找有嵌入式设计或硬件设计基础的人组队，任务完成会更轻松。
3. MSP430F5529 性能较旧，学有余力建议去学 STM32 系列开发。
