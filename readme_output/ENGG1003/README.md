# 工程训练（电子工艺实习）

**课程代码:** ENGG1003

## 建议

1. 虽然项目很简单，但对于无嵌入式和硬件基础的人来说独立完成不容易。
2. 建议找有嵌入式设计或硬件设计基础的人组队，任务完成会更轻松。
3. MSP430F5529 性能较旧，学有余力建议去学 STM32 系列开发。

> 文 / [Gaster]([https://github.com/WDGaster703/](https://github.com/WDGaster703/)), 2025.1

## 课程安排

23 级自动化与电气工程集群的电子工艺实习安排在 2024 年夏季学期。
