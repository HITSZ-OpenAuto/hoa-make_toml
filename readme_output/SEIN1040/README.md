# 中国科技史话

**课程代码:** SEIN1040

## 实验

共两个实验：
1. 实验一为陶艺制作实验，共 6 学时，三节课分别完成塑形、修坯和上釉、装饰，课后老师会帮大家烧制自己塑性修坯上釉的陶器和在烧制好的陶器表面装饰后的陶器。其中第一节塑形是最难的一步，塑形成功之后的两节课就很轻松了。
2. 实验二为机械工艺实验，共 2 学时，老师会带来几十个鲁班锁，自己按照说明书组装，没有复原速度和数量的要求，玩够了就能走。
